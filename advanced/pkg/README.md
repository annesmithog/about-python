# パッケージ

**インストール**
```sh
pip install [package]               # 基本
pip install [package]==[version]    # バージョン指定
pip install [package]==             # インストール可能なバージョンの確認
```

**アンインストール**
```py
pip uninstall [package]
```

**アップデート**
```py
pip install -U [package]
```

**インストールしたパッケージの確認**
```py
pip list
```
