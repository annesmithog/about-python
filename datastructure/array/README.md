# array

## 初期化

```py
from array import array

arr1 = array("b", [0, 1])       # array('b', [0, 1])
arr2 = array("d", [0.1, 0.2])   # array('d', [0.1, 0.2])
arr3 = array("f", [0.1, 0.2])   # array('f', [0.10000000149011612, 0.20000000298023224])
```

## 出力

```py
numbers = array("b", [1, 2, 3])
for number in numbers:
    print(number)
```
